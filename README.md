## Generador de un Servidor de Express

Este generador srvira para  crear servidores a base de express.

# Requirements
* Node 10.15.3
* NPM >= 6.0.0

# Install
```npm install -g generador-proyecto-marco```
# Usage
ejecutamos solo esta orden

```npm-generador-servidor```
